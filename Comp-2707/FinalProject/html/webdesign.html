Introduction to Webdesign

Instructor: Kevin Taylor

Course Content:

  • - HTML and CSS: Basics of creating web pages using HTML and CSS, including syntax, tags, and properties.
  • - Web Page Layout and Design: Design principles, layout techniques, typography, color theory, and accessibility.
  • - Responsive Web Design: Techniques for creating web pages that adapt to different screen sizes and devices.
  • - Web Graphics: Basics of creating and optimizing web graphics, including images and icons.
  • - User Experience and User Interface Design: Designing interfaces that are user-friendly, intuitive, and efficient.
  • - Web Standards and Best Practices: Understanding web standards, validation tools, and best practices for designing websites.
  • - Website Publishing and Hosting: Basics of website publishing and hosting, including domain registration, web servers, and content management systems (CMS).
