Unity: Game Development

Instructor: Jane Doe

Course Content:

  • - Unity Interface: Students will learn about the interface of Unity, which includes Scene view, Hierarchy, Inspector, Game view, Project window, Console, and Toolbar.
  • - Unity Programming: Students will learn C# programming language and how to use it to create custom scripts to control game objects.
  • - 2D and 3D Game Development: Students will learn how to create both 2D and 3D games in Unity, including character design, level design, and animation.
  • - Physics Engine: Unity comes with a built-in physics engine, and students will learn how to use it to create realistic physics-based gameplay mechanics, such as gravity, collision detection, and ragdoll physics.
  • - Artificial Intelligence: Students will learn how to implement simple AI behaviors, such as enemy pathfinding, using Unity's built-in AI tools.
  • - User Interface Design: Students will learn how to design and create user interfaces, including menus, buttons, and sliders, to enhance the player's experience.
  • - Audio Design: Students will learn how to add sound effects and music to their games to create an immersive experience for players.
  • - Optimization: Students will learn how to optimize their games to run smoothly on different devices and platforms, such as desktop, mobile, and VR.
